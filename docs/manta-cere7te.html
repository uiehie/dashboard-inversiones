¿Qué es el Proyecto?
Cere7te
Finacial
Dashboard de Inversiones
Es un software desarrollado en Python + FastAPI que permite gestionar y analizar portafolios de inversión en tiempo real, con datos de mercado actualizados, recomendaciones inteligentes y exportaciones listas para decisión financiera.
Características
  • Validación de usuarios via Login + Registro (JWT)
  • Alta y gestión de acciones al portafolio
  • Cálculo automático de ROI total y por acción
  • Precios en tiempo real con Yahoo Finance API
  • Recomendaciones inteligentes por perfil de riesgo
  • Score de diversificación sectorial automático
  • Historial de transacciones con filtros
  • Alertas de precio configurables por acción
  • Exportación a JSON, CSV y Excel real (.xlsx)
  • Conexión a Base de Datos MySQL
